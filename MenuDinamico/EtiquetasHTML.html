102 Etiquetas HTML

Nombre de la Etiqueta Descripción Ejemplo de Uso Elemento Adicional
<a> Define un hipervínculo. Se utiliza para enlazar una página a otra o un recurso. <a href="https://www.example.com">Visita Ejemplo</a> Se puede agregar el atributo target="_blank" para abrir el enlace en una nueva pestaña.
<p> Define un párrafo de texto. <p>Este es un párrafo.</p> Puedes incluir elementos en línea como <a>, <strong>, <em> dentro del párrafo.
<img> Incrusta una imagen en la página. <img src="imagen.jpg" alt="Descripción de la imagen"> Es importante incluir el atributo alt para accesibilidad y SEO.
<div> Define una sección o contenedor en el documento. <div>Contenido del div</div> Comúnmente se usa para agrupar elementos o aplicar estilos mediante CSS.
<h1> Define el encabezado más importante en la página. <h1>Este es un Encabezado</h1> Los encabezados van desde <h1> (más importante) hasta <h6> (menos importante).
<ul> Define una lista no ordenada (con viñetas). <ul><li>Elemento 1</li><li>Elemento 2</li></ul> Puedes usar <ol> para listas ordenadas.
<ol> Define una lista ordenada (con números). <ol><li>Primer ítem</li><li>Segundo ítem</li></ol> Los elementos de la lista son numerados automáticamente.
<table> Define una tabla en el documento. <table><tr><th>Encabezado</th></tr></table> Se usa junto con <tr> para filas, <td> para celdas y <th> para encabezados.
<form> Define un formulario para la entrada de datos. <form><input type="text"></form> Es utilizado para enviar datos a un servidor.
<input> Define un campo de entrada para formularios. <input type="text"> Puede tener varios tipos como <input type="password">, <input type="email">, etc.
<button> Define un botón que puede ser presionado. <button>Haz clic</button> Se utiliza para ejecutar una acción, como enviar un formulario o ejecutar un script.
<strong> Define texto en negrita, usado para resaltar contenido importante. <strong>Texto en negrita</strong> Es diferente de <b>, ya que semánticamente resalta importancia.
<em> Define texto enfatizado, generalmente en cursiva. <em>Texto enfatizado</em> Se usa para enfatizar el contenido, mejorando la accesibilidad.
<blockquote> Define una cita en bloque, usualmente de otro autor. <blockquote>Esta es una cita.</blockquote> Se puede acompañar con el atributo cite para referenciar la fuente.
<br> Inserta un salto de línea, útil para separar texto. <p>Primer párrafo<br>Segundo párrafo</p> Es un elemento vacío, no requiere etiqueta de cierre.
<code> Define un fragmento de código fuente en el documento. <code>console.log('Hola Mundo');</code> Se usa para resaltar código dentro de un texto.
<sub> Define texto en subíndice. <math>H<sub>2</sub>O</math> Usado comúnmente en fórmulas químicas y matemáticas.
<sup> Define texto en superíndice. <math>x<sup>2</sup></math> Se utiliza para exponer exponentes en expresiones matemáticas.
<hr> Define una línea horizontal que se usa para separar contenido. <hr> Es un elemento vacío, no requiere etiqueta de cierre.
<span> Define una sección en línea sin afectar el flujo del documento. <span>Texto dentro de un span</span> Se usa principalmente para aplicar estilos sin alterar la estructura.
<label> Define una etiqueta para un control de formulario. <label for="nombre">Nombre:</label><input id="nombre" type="text"> El atributo "for" asocia la etiqueta con un campo de entrada específico.
<fieldset> Define un grupo de controles en un formulario, con una caja de borde. <fieldset><legend>Información Personal</legend></fieldset> Se utiliza para agrupar elementos relacionados en un formulario.
<legend> Define una leyenda para un <fieldset>, proporciona una descripción del grupo. <fieldset><legend>Detalles de contacto</legend></fieldset> Mejora la accesibilidad al describir el propósito del campo.
<iframe> Define un marco en línea que puede contener otra página web. <iframe src="https://www.example.com"></iframe> Puede ser utilizado para incrustar videos, mapas o cualquier otra página web.
<audio> Define un reproductor de audio. <audio controls><source src="audio.mp3" type="audio/mp3"></audio> El atributo "controls" añade controles como play, pausa y volumen.
<video> Define un reproductor de video. <video controls><source src="video.mp4" type="video/mp4"></video> El atributo "controls" permite al usuario controlar la reproducción del video.
<textarea> Define un área de texto multilinea en un formulario. <textarea rows="4" cols="50">Texto aquí</textarea> Los atributos "rows" y "cols" definen el tamaño del área de texto.
<select> Define un menú desplegable en un formulario. <select><option value="1">Opción 1</option></select> El atributo "multiple" permite seleccionar múltiples opciones.
<option> Define una opción dentro de un <select> o <optgroup>. <select><option>Opción 1</option></select> Se utiliza dentro de un <select> o <optgroup> para crear un menú desplegable.
<optgroup> Define un grupo de opciones dentro de un menú desplegable. <select><optgroup label="Grupo 1"><option>Opción 1</option></optgroup></select> El atributo "label" define el nombre del grupo.
<link> Define una relación entre el documento actual y un recurso externo, generalmente usado para vincular a hojas de estilo. <link rel="stylesheet" href="styles.css"> El atributo "rel" define la relación entre el documento y el recurso.
<meta> Define metadatos sobre el documento HTML, como la codificación de caracteres o la descripción de la página. <meta charset="UTF-8"> Se utiliza dentro del elemento <head>.
<title> Define el título de la página, que aparece en la barra de título o en la pestaña del navegador. <title>Mi página web</title> Es obligatorio para cualquier documento HTML.
<style> Define estilos CSS internos para la página. <style>body {background-color: lightblue;}</style> Usualmente se coloca dentro del <head> para aplicar estilos directamente en el documento.
<script> Define un bloque de código JavaScript en la página. <script>alert('Hola Mundo');</script> Se puede colocar dentro del <head> o al final del <body>.
<noscript> Define un bloque de contenido alternativo para navegadores que no soportan JavaScript. <noscript>Este navegador no soporta JavaScript.</noscript> Se utiliza para mostrar un mensaje cuando JavaScript está deshabilitado en el navegador del usuario.
<head> Define la sección de metadatos del documento HTML, como el título y enlaces a hojas de estilo. <head><title>Título de la página</title></head> Debe incluirse solo una vez por documento, al principio del archivo HTML.
<body> Define el contenido visible de la página web. <body>Contenido de la página</body> Es la parte central del documento donde se coloca el contenido que el usuario verá.
<html> Define el documento HTML en sí mismo. <html><head></head><body></body></html> Es el elemento raíz de cualquier documento HTML.
<meta> Define metadatos, como la codificación de caracteres y descripción de la página. <meta charset="UTF-8"> Normalmente se coloca dentro del <head> para establecer la codificación de caracteres.
<style> Define los estilos CSS directamente en el documento HTML. <style>body { color: blue; }</style> Los estilos se aplican a los elementos dentro del documento.
<svg> Define un contenedor para gráficos vectoriales escalables (SVG). <svg width="100" height="100"><circle cx="50" cy="50" r="40" stroke="green" stroke-width="4" fill="yellow"/></svg> Se utiliza para crear gráficos interactivos y animados dentro de una página web.
<canvas> Define un área de dibujo para gráficos dinámicos con JavaScript. <canvas id="myCanvas" width="200" height="200"></canvas> Se usa con JavaScript para crear gráficos, juegos y otros elementos visuales interactivos.
<datalist> Define un conjunto de opciones predefinidas para los controles de entrada de datos. <input list="browsers"><datalist id="browsers"><option value="Chrome"><option value="Firefox"></datalist> El atributo "list" se asocia con un control de entrada como <input> para proporcionar opciones predefinidas.
<meter> Define un medidor visual de un valor dentro de un rango conocido. <meter value="0.7"></meter> Se usa comúnmente para representar valores como el progreso, la temperatura o la carga de batería.
<progress> Define una barra de progreso. <progress value="50" max="100"></progress> Se usa para mostrar el progreso de una tarea, como la carga de un archivo.
<time> Define una fecha o hora. <time datetime="2025-02-07">7 de febrero de 2025</time> Es útil para especificar fechas, horas o duraciones en formatos estándar.
<mark> Define texto resaltado. <mark>Texto resaltado</mark> Se usa comúnmente para destacar palabras en los resultados de búsqueda.
<var> Define una variable en un contexto matemático o de programación. <var>x = 5</var> Se utiliza en el contexto de matemáticas y programación para representar variables.
<kbd> Define texto que representa entrada del teclado. <kbd>Ctrl + C</kbd> Usado para mostrar combinaciones de teclas o comandos de teclado.
<samp> Define una muestra de salida del sistema. <samp>Error: 404 Not Found</samp> Comúnmente usado para mostrar mensajes de error o salida de consola.
<bdo> Define la dirección del texto. <bdo dir="rtl">Texto de derecha a izquierda</bdo> Se usa para cambiar la dirección del texto, útil para idiomas como el árabe o hebreo.
<wbr> Define una recomendación para un salto de línea en una palabra. <wbr>Intervención en<wbr> palabra</wbr> Se utiliza en palabras largas para sugerir puntos donde se puede romper el texto.
<small> Define texto en un tamaño más pequeño. <small>Este es un texto pequeño</small> Se usa para resaltar información adicional, como notas al pie.
<big> Define texto en un tamaño más grande. <big>Este es un texto grande</big> Puede ser útil para resaltar palabras clave o encabezados pequeños.
<b> Define texto en negrita, generalmente utilizado para resaltar sin significar importancia. <b>Texto en negrita</b> No tiene un significado semántico, solo estiliza el texto.
<i> Define texto en cursiva. <i>Texto en cursiva</i> Usado para mostrar énfasis o para nombres de títulos, libros, etc.
<u> Define texto subrayado. <u>Texto subrayado</u> Usado para resaltar texto, aunque no es semántico.
<del> Define texto que ha sido eliminado. <del>Este texto ha sido eliminado</del> Generalmente se utiliza en conjunto con la etiqueta <ins> para mostrar cambios en un documento.
<ins> Define texto que ha sido insertado. <ins>Este texto ha sido insertado</ins> Se utiliza para mostrar texto añadido a un documento.
<cite> Define una referencia a una fuente de cita. <cite>Fuente:</cite> Usado para citar fuentes de información o citas de autores.
<abbr> Define una abreviatura o acrónimo. <abbr title="Hypertext Markup Language">HTML</abbr> El atributo "title" se usa para mostrar la descripción completa al pasar el mouse sobre la abreviatura.
<address> Define información de contacto, generalmente relacionada con un autor o propietario de un documento. <address>Contacto: info@example.com</address> Comúnmente se usa para mostrar direcciones de correo electrónico, números de teléfono, etc.
<area> Define una área dentro de una imagen map (mapa de imagen). <map name="mapa"><area shape="rect" coords="34,44,270,350" href="https://example.com" alt="Ejemplo" /></map> Se usa con la etiqueta <map> para crear zonas clicables dentro de una imagen.
<article> Define contenido independiente que puede ser distribuido o reutilizado. <article>Este es un artículo de ejemplo</article> Ideal para blogs, noticias o entradas de cualquier tipo de contenido autónomo.
<aside> Define contenido que está relacionado, pero no es parte del contenido principal. <aside>Contenido adicional o relacionado</aside> Comúnmente se usa para mostrar barras laterales, citas o enlaces relacionados.
<audio> Define contenido de audio, como música o grabaciones. <audio controls><source src="audio.mp3" type="audio/mp3"></audio> El atributo "controls" agrega controles para reproducir, pausar y ajustar el volumen del audio.
<bdi> Define un bloque de texto que debe ser tratado como independiente respecto al texto circundante. <bdi>Texto independiente</bdi> Utilizado para contener texto que puede tener una dirección diferente al contenido circundante.
<bdo> Permite controlar la dirección del texto (por ejemplo, derecha a izquierda o izquierda a derecha). <bdo dir="rtl">Texto de derecha a izquierda</bdo> Útil para idiomas como árabe o hebreo.
<br> Inserta un salto de línea. <p>Este es un párrafo con un salto de línea<br>Este es el texto en la nueva línea</p> Se usa para separar líneas de texto en lugar de crear nuevos párrafos.
<button> Define un botón interactivo. <button>Haz clic</button> Se utiliza en formularios y páginas interactivas para capturar la acción del usuario.
<canvas> Define un área de dibujo para gráficos dinámicos con JavaScript. <canvas id="myCanvas" width="200" height="200"></canvas> Usado para crear gráficos, juegos o cualquier tipo de contenido visual interactivo.
<caption> Define un título para una tabla. <table><caption>Título de la tabla</caption><tr><td>Contenido</td></tr></table> Generalmente se usa para describir la tabla de manera breve y concisa.
<cite> Define una cita de una fuente. <cite>Fuente: John Doe</cite> Se utiliza para atribuir citas de libros, artículos o investigaciones.
<code> Define un fragmento de código de programación. <code>let x = 10;</code> Usado para resaltar el código en los tutoriales o ejemplos de programación.
<col> Define propiedades para las columnas dentro de una tabla. <col span="2"> Se usa en conjunto con la etiqueta <colgroup> para aplicar estilos a columnas específicas de la tabla.
<colgroup> Define un grupo de columnas dentro de una tabla. <colgroup><col span="2"></colgroup> Se utiliza para aplicar estilos o propiedades a un grupo de columnas en una tabla.
<data> Asocia un valor con contenido de texto. <data value="2025-02-07">7 de febrero de 2025</data> Ideal para marcar información con datos semánticos adicionales, como fechas o valores.
<del> Define texto que ha sido eliminado de un documento. <del>Texto eliminado</del> Usado generalmente en combinación con <ins> para mostrar cambios en un documento.
<details> Define un bloque de contenido que el usuario puede expandir o contraer. <details><summary>Más detalles</summary>Contenido adicional</details> Se usa para ocultar o mostrar contenido adicional, como una lista de preguntas frecuentes.
<dfn> Define un término que se está definiendo. <dfn>HTML</dfn> Comúnmente usado para resaltar un término técnico o específico que se está definiendo.
<dialog> Define un cuadro de diálogo o ventana emergente. <dialog open>Este es un cuadro de diálogo</dialog> Usado para mostrar mensajes, formularios o notificaciones interactivas dentro de la página.
<div> Define un contenedor genérico. <div>Contenido del div</div> Comúnmente usado para aplicar estilos CSS a bloques de contenido.
<dl> Define una lista de definiciones. <dl><dt>HTML</dt><dd>HyperText Markup Language</dd></dl> Se usa para crear diccionarios o listas de términos y sus definiciones.
<dt> Define un término dentro de una lista de definiciones. <dl><dt>HTML</dt><dd>HyperText Markup Language</dd></dl> Se usa dentro de la etiqueta <dl> para definir un término que será explicado o definido en <dd>.
<em> Define texto enfatizado, generalmente se representa en cursiva. <p><em>Este texto está enfatizado</em></p> Se utiliza para enfatizar el contenido de una manera semántica.
<embed> Define un recurso embebido como multimedia (audio, video, etc.) en la página. <embed src="video.mp4" type="video/mp4" width="400" height="300"> Se usa para integrar contenido multimedia externo como videos o audios en la página.
<fieldset> Define un grupo de controles en un formulario. <fieldset><legend>Información Personal</legend></fieldset> Usado para agrupar elementos relacionados en un formulario, mejorando la accesibilidad.
<figcaption> Define un título o descripción para una imagen o figura. <figure><img src="image.jpg"><figcaption>Descripción de la imagen</figcaption></figure> Se utiliza dentro de la etiqueta <figure> para proporcionar una leyenda o texto descriptivo.
<figure> Define contenido autónomo como imágenes, ilustraciones o diagramas. <figure><img src="image.jpg"><figcaption>Imagen de ejemplo</figcaption></figure> Comúnmente se usa para agrupar una imagen con su respectiva leyenda o descripción.
<font> Define el tipo, tamaño y color de la fuente (desactualizado, mejor usar CSS). <font color="red">Texto en rojo</font> Aunque es obsoleta, todavía se utiliza en algunos casos para aplicar estilos básicos de texto.
<footer> Define el pie de página de un documento o sección. <footer><p>Derechos reservados 2025</p></footer> Se usa para incluir información de copyright, enlaces relacionados o información de contacto.
<form> Define un formulario HTML que recoge datos del usuario. <form action="/submit" method="post"><input type="text" name="name"></form> Se usa para crear formularios interactivos que envían datos a un servidor.
<h1> Define el encabezado más importante. <h1>Encabezado principal</h1> Se utiliza para jerarquizar el contenido, siendo <h1> el más relevante y <h6> el menos importante.
<h2> Define el segundo nivel de encabezado. <h2>Subtítulo del contenido</h2> Es común utilizar <h2> para títulos de secciones secundarias dentro de la página.
<h3> Define el tercer nivel de encabezado. <h3>Subsección del contenido</h3> Se usa para dividir subsecciones dentro de las secciones.
<h4> Define el cuarto nivel de encabezado. <h4>Título de la subsección</h4> Usado para contenido más detallado dentro de una subsección.
<h5> Define el quinto nivel de encabezado. <h5>Título menor del contenido</h5> Utilizado en contenido más específico dentro de una subsección.
<h6> Define el sexto nivel de encabezado. <h6>Subtítulo menos importante</h6> Es el encabezado menos importante y se usa para subdivisiones más finas.
<head> Define el encabezado de un documento HTML, contiene metadatos, enlaces a recursos, etc. <head><title>Título de la página</title></head> Es fundamental para incluir elementos como el título de la página, enlaces a archivos CSS o scripts JS.
<header> Define la cabecera de un documento o sección. <header><h1>Bienvenido a mi página</h1></header> Se usa para contener elementos introductorios o de navegación dentro de una sección.
<hr> Inserta una línea horizontal, comúnmente usada para separar contenido. <hr> Es una forma de dividir visualmente el contenido sin necesidad de usar elementos adicionales.
<i> Define texto en cursiva. <p><i>Este texto está en cursiva</i></p> Comúnmente se usa para resaltar términos, nombres o títulos en un texto.
<iframe> Define un marco (frame) para mostrar otro documento dentro del documento actual. <iframe src="https://www.example.com"></iframe> Se usa para insertar contenido externo como un video o una página web dentro de la página actual.
<img> Define una imagen en la página web. <img src="image.jpg" alt="Descripción de la imagen"> Es necesario incluir el atributo "alt" para describir la imagen en caso de que no se cargue.
<input> Define un campo de entrada para que el usuario introduzca datos en un formulario. <input type="text" name="nombre"> El atributo "type" puede ser de diferentes tipos como text, password, email, etc.
<ins> Define un texto insertado, generalmente se muestra subrayado. <ins>Texto insertado</ins> Es usado para indicar texto que ha sido añadido a un documento.
<kbd> Define texto que representa la entrada del teclado. <kbd>Ctrl + C</kbd> Es útil para representar combinaciones de teclas.

<legend> Define un título para un elemento <fieldset>. <fieldset><legend>Información Personal</legend></fieldset> Usado para dar contexto a los elementos agrupados dentro de <fieldset>.
<li> Define un elemento de una lista. <ul><li>Elemento de lista</li></ul> Usado dentro de listas ordenadas (<ol>) o no ordenadas (<ul>).
<link> Define una relación entre el documento y un recurso externo, como una hoja de estilo. <link rel="stylesheet" href="styles.css"> Se usa comúnmente para vincular hojas de estilo CSS o fuentes externas.
<main> Define el contenido principal de un documento, que debe ser único y relevante para la página. <main><h1>Contenido principal</h1></main> Mejora la accesibilidad al identificar claramente la parte principal de la página.
